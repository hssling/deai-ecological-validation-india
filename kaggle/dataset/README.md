# Kaggle Dataset: DEAI Healthy Ageing India LASI Wave 1

This Kaggle dataset provides the public release for the DEAI ecological validation manuscript.

## Included materials

- processed state-level tables used in the manuscript
- publication figures
- manuscript submission assets
- reproducibility scripts
- public notebook support files

## Scientific boundary

DEAI is a state-level ecological surveillance index. It is not:

- an individual-level risk score
- a biological-age clock
- a causal inference model

## Main outputs

- `deai_real_state_rankings.csv`
- `deai_real_outcome_correlations.csv`
- `deai_real_robustness.csv`
- `deai_component_diagnostics.csv`
- manuscript figures 1 to 4

## Source data note

This release contains derived outputs based on publicly available aggregate sources. Original public reports remain with LASI/IIPS and related providers.
