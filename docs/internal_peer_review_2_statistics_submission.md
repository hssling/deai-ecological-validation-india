# Internal Peer Review 2 - Statistics, Reliability, and JIAG Submission Readiness

## Overall Recommendation

Potentially suitable for JIAG after tightening the article type, reducing overclaiming, and preparing journal-specific files.

## Major Comments

1. **Article framing should match JIAG readership.**  
   A methods-heavy "AI biological ageing" framing may be too broad. The stronger framing is: a reproducible digital exposome index for healthy-ageing surveillance in India, with LASI state-level ecological validation.

2. **Multiple testing must be transparent.**  
   Seven outcome correlations were tested. Only the multimorbidity association survives FDR correction. Mortality should be interpreted as nominal, stable, and hypothesis-supporting.

3. **Reliability evidence should be shown.**  
   The manuscript should include internal consistency and component diagnostics. Cronbach alpha=0.688 is acceptable for a deliberately multi-domain exposome index, but it is not high enough to imply a single latent construct.

4. **Submission files should be separated.**  
   JIAG-style package should include title page, blinded manuscript, cover letter, declarations, contributor details, figure file map, and submission checklist.

5. **Blinding must be preserved.**  
   The blinded manuscript should not include author names, institutional affiliations, local folder paths, ethics numbers, ORCID, or acknowledgments that identify authors.

## Minor Comments

- Keep the abstract within approximately 250 words.
- Prefer Vancouver-style numbered references in the submission manuscript.
- Keep display items limited and move extended tables to supplementary files.
- Mention AI/modeling methods in plain geriatric-public-health language.
